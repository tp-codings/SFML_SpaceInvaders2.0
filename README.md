# Space Invaders 2.0

A little Space Invaders game based on SFML. 

## Features
- move spacecraft
- fire on enemies

# Getting Started

1. Clone Repository
`git clone <https://github.com/tp-codings/SFML_SpaceInvaders2.0.git`>
2. Build Executable (e.g. Visual Studio)

# Usage

Movement:

- W: Up
- A: Left
- S: Down
- D: Right
- Space: Fire
